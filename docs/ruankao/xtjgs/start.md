# 考试说明

## 科目

1. 科目一：综合知识。  
  客观题：75道  150分钟  
  特点：范围广，但是考试时间充足。
2. 科目二：案例分析。  
  主观题：5道大题（只答3道）  90分钟
3. 科目三：论文写作。  
  论问题：4道题选做1道  120分钟
